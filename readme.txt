== Description ==

BetterImages offers a better way of generating images code side, for WordPress.

This plugin is aimed at developers building WordPress themes that are looking for a better way of generating images at
various sizes.

See [README.md](https://github.com/ethercreative/BetterImages/blob/master/README.md)


== Installation ==

Upload and activate.
